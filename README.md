Developed a full-stack real estate marketplace that allows users to search, compare, and evaluate properties for sale or rent through a responsive and intuitive interface. The application was built with Next.js and Styled Components on the frontend, and Node.js with Express on the backend, with a RESTful API enabling efficient communication across the stack. To improve property discovery, I integrated Leaflet for interactive map-based location visualization and the Google Places API for smarter, location-aware search.

## Backend
